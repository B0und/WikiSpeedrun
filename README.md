## Build it yourself

Clone the project:

```
git clone https://github.com/B0und/WikiSpeedrun.git
```

In the project directory you can run `pnpm i` and then:

### `pnpm run dev`

To run the app in the development mode.

### `pnpm run build`

To build the app for production to the `build` folder.
